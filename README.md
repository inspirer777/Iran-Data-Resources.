# Iran Data Resources

این مخزن شامل لینک‌های مفید به داده‌ها و اطلاعات آماری و هواشناسی ایران است.

## منابع داده‌ها

### １- داده‌های درجه هوای کل ایستگاه‌های کشور ایران
- [Ogimet Ranking of Weather Parameters](http://ogimet.com/ranking.phtml.en)
- [Ogimet Synoptic Data for Iran](http://ogimet.com/cgibin/gsynext?lang=en&state=Iran&rank=100&ano=2017&mes=01&day=30&hora=12&Send=send)

### ２- داده‌های ماهانه و سالانه تعداد روزهای همراه با بارندگی برای کل شهرهای کشور
- [Iran Hydrology](http://www.iranhydrology.net/meteo.asp)

### ３- داده‌های ایستگاه سینوپتیک تا سال 2010 برای ایران
- [Synoptic Stations Archive](http://www.chaharmahalmet.ir/iranarchive.asp)

### ４- آرشیو شاخص کیفیت هوای تهران
- [Tehran AQI Archive](http://airnow.tehran.ir/home/AQIArchive.aspx)

### ５- بانک مرکزی – آمارها و داده‌ها
- [Central Bank of Iran Statistics](http://www.cbi.ir/section/1378.aspx)

### ６- نتایج سرشماری عمومی نفوس و مسکن – مرکز آمار ایران
- [Census Results - Iran Statistics Center](https://www.amar.org.ir/سرشماری-عمومی-نفوس-و-مسکن/نتایج-سرشماری)
